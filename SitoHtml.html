Cos'è PixiJS

PixiJS è un sistema di rendering open source web-based che offre prestazioni velocissime per giochi, visualizzazione dati e altri progeti ad alta intensità grafica. Utilizza WebGL, una API (Application Programming Interface) JavaScript di grafica 2D e 3D sul browser, per visualizzare immagini e altri contenuti 2D. PixiJS rimpiazza Flash nel mondo moderno di HTML5 e, provvede migliori performance e effetti pixel che Flash non poteva raggiungere. Perfetto per giochi online, contenuti educativi, pubblicità interattive, visualizzazione delle informazioni, ...per ogni web-based applicationi che richiede una grafica complessa.
